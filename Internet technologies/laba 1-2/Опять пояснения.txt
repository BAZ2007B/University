<!DOCTYPE HTML> - это объявление типа документа.

<html> - открывающий тег элемента HTML.

<head> - открывающий тег секции заголовка документа.

<title>Lyrics</title> - устанавливает заголовок документа на "Lyrics".

<meta charset="utf-8"> - устанавливает кодировку страницы на UTF-8.

<meta name="keywords" content="The Devil In I, lyrics, Slipknot"> - задает метатег "keywords" со списком ключевых слов.

<meta name="description" content="The Devil In I lyrics"> - задает метатег "description" с описанием страницы.

<style> - открывающий тег для определения стилей CSS.

@import url('https://fonts.googleapis.com/css2?family=Play&display=swap'); - импортирует шрифт "Play" из Google Fonts.

@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+HK&display=swap'); - импортирует шрифт "Noto Sans HK" из Google Fonts.

</style> - закрывающий тег для определения стилей CSS.

<link href="styles/style.css" rel="stylesheet" type="text/css"/> - подключает CSS-файл для основных стилей страницы.

<link href="styles/songTitle.css" rel="stylesheet" type="text/css"/> - подключает CSS-файл для стилей заголовков песен.

</head> - закрывающий тег секции заголовка документа.

<body> - открывающий тег для тела документа.

<div class="trash"> - открывающий тег для блока "trash".

<button class="button" type="button">^_^</button> - создает кнопку с классом "button" и текстом "^_^".

</div> - закрывающий тег для блока "trash".

<div class="content"> - открывающий тег для блока "content".

<div class="song"> - открывающий тег для блока "song".

<div class="gridElement1"> - открывающий тег для первого элемента сетки.

<picture> - создает контейнер для элементов изображения.

<source src="images/cover.png"> - указывает источник для изображения.

<img id="image1" src="images/cover.png"> - устанавливает изображение, которое будет отображаться, если браузер не поддерживает элемент <picture>.

</picture> - закрывающий тег для контейнера изображения.

</div> - закрывающий тег для первого элемента сетки.

<div class="gridElement2"> - открывающий тег для второго элемента сетки

<h1> - открывающий тег заголовка первого уровня.
<a> - открывающий тег ссылки.
class="text" - класс ссылки, используется для применения стилей.
href="https://www.youtube.com/watch?v=XEEasR7hVhA" - атрибут ссылки, указывающий на адрес видео на YouTube.
The Devil In I - текст, отображаемый в ссылке на видео.
<span> - открывающий тег элемента строчного текста.
class="dot" - класс элемента, используется для применения стилей.
. - текст, отображаемый в элементе span, в данном случае это точка.
</span> - закрывающий тег элемента span.
</a> - закрывающий тег ссылки.
</h1> - закрывающий тег заголовка первого уровня.
<h2> - открывающий тег заголовка второго уровня.
<a> - открывающий тег ссылки.
class="text" - класс ссылки, используется для применения стилей.
href="https://www.youtube.com/channel/UCOJZ1tna8yj8mAEITPkHNCQ" - атрибут ссылки, указывающий на канал Slipknot на YouTube.
Slipknot - текст, отображаемый в ссылке на канал.
</a> - закрывающий тег ссылки.
</h2> - закрывающий тег заголовка второго уровня.
</div> - закрывающий тег блока div, содержащего информацию о песне и ссылки на видео и канал.
