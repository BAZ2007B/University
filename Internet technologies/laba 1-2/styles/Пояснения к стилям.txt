html::selection - это псевдоэлемент, который используется для определения стилей выделенного текста на веб-странице.
background-color: white; - задает белый цвет фона для выделенного текста.
color: #fff; - задает белый цвет текста для выделенного текста.
html - это селектор для элемента html на странице.
background-color: #4E5754; - задает серый цвет фона для всей страницы.
body - это селектор для элемента body на странице.
width: calc(100% - 60%); - задает ширину элемента body, вычитая 60% от ширины экрана.
height: auto; - задает автоматическую высоту для элемента body.
margin: 1em auto; - задает отступы от верхнего и нижнего краев элемента body, и автоматически центрирует элемент по горизонтали.
padding: 1em 3em; - задает внутренние отступы для элемента body.
background-color: #293133; - задает темно-зеленый цвет фона для элемента body.
border: 5px solid #B5B8B1; - задает пятимерную рамку вокруг элемента body с серым цветом.
h1,h2,h3,h6,p,ul,a,span - это селектор для всех заголовков, параграфов, списков, ссылок и спанов на странице.
margin: 0; - задает нулевые отступы для всех элементов, которые были выбраны в предыдущей строке.
.content - это класс, который будет применен к элементам с классом "content".
font-family: 'Play', sans-serif; - задает шрифт "Play" для элементов с классом "content".
font-size: 1.5em; - задает размер шрифта в 1,5 раза больше, чем шрифт по умолчанию для элементов с классом "content".
color: #C6C3B5; - задает серый цвет текста для элементов с классом "content".
.song - это класс, который будет применен к элементам с классом "song".
display: grid; - устанавливает элементы с классом "song" в сетку.
gap: 0.5em; - задает отступы между элементами в сетке для элементов с классом "song".
grid-template-columns: repeat(2, 1f); - задает два столбца в сетке для элементов с классом "song".
.lyrics {text-align: left;} - задает выравнивание текста в блоке с классом "lyrics" по левому краю.
h6 {margin-top: 1em;} - задает верхний отступ в один em для заголовка h6.
.more {margin-top: 4em; text-align: left;} - задает верхний отступ в 4 em и выравнивание текста по левому краю для блока с классом "more".
.video {margin-top: 1em; border: none; height: 40vh; width: 40vw;} - задает верхний отступ в один em, отсутствие границы, высоту в 40% высоты окна и ширину в 40% ширины окна для элемента с классом "video".
.trash {margin-top: 2em; text-align: center;} - задает верхний отступ в 2 em и выравнивание текста по центру для блока с классом "trash".
.button {color: white; display: block; position: relative; z-index: -1; padding: 0.7em 1em 1em 1em; border: 0; border-radius: 10px; background-image: linear-gradient(to top left, rgba(0, 0, 0, 0.2), rgba(0, 0, 0, 0.2) 30%, rgba(0, 0, 0, 0)); box-shadow: inset 2px 2px 3px rgba(255, 255, 255, 0.6), inset -2px -2px 3px rgba(0, 0, 0, 0.6); animation-name: rgb; animation-duration: 4s; animation-direction: alternate-reverse; animation-iteration-count: infinite; transition: border-radius 2s, transform 5s;} - задает стили для кнопки: белый цвет текста, блочный элемент, относительное позиционирование, z-index равный -1, отступы, отсутствие границы, скругление углов равное 10px, задний фон в виде линейного градиента, тень внутри элемента, анимацию смены цвета фона и переходы.
@keyframes rgb {from {background-color: blue;} to {background-color: orange;}} - определяет ключевые кадры для анимации "rgb", где начальный цвет фона кнопки синий, а конечный - оранжевый.
.button:hover {border-radius: 20px; transform: rotate(360deg)} - задает стили для состояния наведения на кнопку: увеличенное скругление углов равное 20px и поворот на 360 градусов.
